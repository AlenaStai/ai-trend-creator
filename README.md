# AI Trend Creator

Инструмент, который закрывает всю цепочку от тренда до готового короткого видео:
**Trends → Analyze → Generate Content → Generate Video → Export**.

## Проблема

Автору AI-блога нужно регулярно выпускать контент про нейросети, который резонирует
с актуальной повесткой — а для этого нужно успевать следить за трендами в TikTok,
Instagram и YouTube и быстро превращать их в сценарий и видео. Вручную это долго:
искать тренды по разным площадкам, оценивать, какой из них действительно выстрелит,
писать вирусный сценарий под свой стиль и потом ещё собирать видео. AI Trend Creator
автоматизирует всю эту цепочку одним инструментом, оставляя автору только творческие
решения — какой тренд выбрать и как подать.

## Пользователь

Автор блога/канала про нейросети (в первую очередь — под конкретный Telegram-канал
про AI: применение нейросетей в жизни, в работе и заработок на них), который выпускает
короткие видео и хочет опираться на актуальные тренды, но не имеет времени вручную
мониторить несколько соцсетей и площадок.

## Основная функция

**Вход:**
- Ничего вручную собирать не нужно — сервис сам подтягивает актуальные тренды
  из Google Trends, YouTube, TikTok и Reddit по одному нажатию кнопки
- Либо: собственная идея автора (текстом или голосовой заметкой), в обход анализа
  трендов

**Что делает сервис:**
1. **Trends** — собирает единую ленту трендов из нескольких источников и переводит
   заголовки на русский
2. **Analyze** — оценивает через Claude API вероятность вирусности каждого тренда
   («вероятно залетит» / «под вопросом» / «вряд ли залетит»), релевантность нише
   и уровень конкуренции; по запросу — уточняет через веб-поиск суть тренда и дату
3. **Generate Content** — генерирует вирусный сценарий под авторский стиль (хук
   в первые 2 секунды, короткие фразы, самоирония, без инфоблогерских клише), caption
   и image-промпты; сценарий можно вручную отредактировать
4. **Generate Video** — превращает сценарий в готовый ролик через HeyGen Video Agent:
   визуал, аватар, монтаж, темп — одним вызовом
5. **Export** — собирает видеофайл и текстовый пакет в единую папку

**Инструменты в цепочке:** Trends MCP + YouTube Data API/Reddit API (сбор трендов),
Claude API — Opus и Haiku (скоринг, сценарий, перевод, веб-поиск), HeyGen CLI Video
Agent (видео), SpeechRecognition (голосовая идея).

## Формат результата

На выходе — готовый пакет для публикации:
- видеофайл (портретный формат, под Shorts/Reels) с аватаром автора
- сценарий с хуком и разметкой пауз
- caption для соцсети
- image-промпты

Публикация в соцсети сознательно вне цепочки — дальше автор публикует сама, где
и как захочет.

## MVP и планы на будущее

**Реализовано (MVP):**
- сбор трендов из нескольких источников с фолбэком, если основной недоступен
- скоринг трендов с оценкой вирусности и обоснованием через Claude API
- генерация вирусного сценария под авторский стиль, включая быстрый режим
  по собственной идее (текст/голос)
- генерация готового видео целиком через HeyGen Video Agent
- экспорт финального пакета (видео + тексты)

**Вне MVP, в планах на будущее:**
- автопостинг готового видео в разные соцсети (YouTube Shorts, Instagram Reels,
  TikTok, Telegram)
- аналитика по результатам уже опубликованного контента

## Технологии

- Python 3.11+ (проверено и работает также на 3.9)
- Streamlit — UI
- SQLite — хранение трендов (`data/trends.db`)
- Claude API (Anthropic) — скоринг трендов, генерация сценариев, перевод заголовков
  (Haiku) и уточнение сути тренда через встроенный веб-поиск (Opus); ответы в JSON
- Trends MCP — основной источник трендов (Google Trends, YouTube, TikTok, Reddit)
- YouTube Data API / Reddit API — фолбэк, если Trends MCP недоступен
- HeyGen CLI (Video Agent) — генерация готового видео с аватаром
- SpeechRecognition — распознавание голосовой заметки (бесплатно, без ключа)

## Установка

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Переменные окружения

```bash
cp .env.example .env
```

Заполните `.env`:

| Переменная | Откуда взять | Обязательна |
|---|---|---|
| `ANTHROPIC_API_KEY` | console.anthropic.com | Да — без неё не работают Analyze и Generate Content |
| `TRENDS_MCP_API_KEY` | trendsmcp.ai (бесплатно, 100 запросов/мес) | Да — основной источник трендов |
| `YOUTUBE_API_KEY` | Google Cloud Console → включить «YouTube Data API v3» | Нет — фолбэк-источник |
| `REDDIT_CLIENT_ID` / `REDDIT_CLIENT_SECRET` | reddit.com/prefs/apps, тип приложения «script» | Нет — фолбэк-источник |
| `HEYGEN_AVATAR_ID` / `HEYGEN_VOICE_ID` | см. ниже | Да — без них не работает Generate Video |

### HeyGen (видео)

Ключ API не нужен — авторизация через OAuth-сессию CLI:

```bash
curl -fsSL https://static.heygen.ai/cli/install.sh | bash
heygen auth login --oauth
```

Аватар создаётся один раз, вручную, через веб-интерфейс HeyGen
(Avatars → Create New Avatar → Digital Twin — нужна короткая запись себя на видео,
от 2 минут, плюс отдельное consent-видео до 30 секунд, платформа сама подскажет текст).
После создания найдите `avatar_id` и `voice_id` (`heygen avatar looks list`) и впишите
в `.env`.

## Запуск

```bash
streamlit run app.py
```

Откроется на `http://localhost:8501`, 4 вкладки по шагам цепочки.

## Как пользоваться

1. **Trends** — кнопка «Собрать тренды». Пробует Trends MCP, при ошибке или пустом
   ответе автоматически переключается на YouTube, затем на Reddit. Тренды сохраняются
   в SQLite, заголовки сразу переводятся на русский (колонка «Заголовок (рус.)»).
   Повторный сбор не плодит дубли — существующий тренд просто обновляется.
2. **Analyze** — выберите тренд из списка (можно по русскому переводу), «Проанализировать».
   Claude API оценивает вероятность вирусности («вероятно залетит» / «под вопросом» /
   «вряд ли залетит»), релевантность нише, конкуренцию и причину тренда. Кнопка
   «🔎 Найти дату и суть (веб-поиск)» — отдельно, по запросу: Claude реально гуглит,
   что стоит за трендом и когда он завирусился (медленнее и расходует больше токенов,
   поэтому не запускается автоматически).
3. **Generate Content** — «Сгенерировать сценарий» на основе последнего анализа.
   Возвращает хук, полный сценарий (с явными паузами `[пауза]` под темп видео), caption
   и image-промпты — в авторском стиле (короткие фразы, самоирония, без инфоблогерских
   CTA). Готовый текст можно поправить вручную в полях ниже перед отправкой в HeyGen.
   Если оценка Claude на Analyze не совпадает с твоим чутьём — можно пропустить анализ
   и сгенерировать сценарий сразу по своей идее (текстом или голосом через 🎤).
4. **Generate Video** — «Отправить сценарий в HeyGen». Рендер занимает **20-45 минут**,
   поэтому отправка и проверка статуса разделены: жмите «Проверить статус», пока не
   увидите `completed`, затем «Скачать готовое видео» (сохраняется в `data/videos/`)
   и «⬇️ Скачать видео на компьютер» — второй вариант отдаёт файл через браузер прямо
   в Загрузки.
5. **Экспорт** (там же, внизу вкладки Video) — «Экспортировать пакет» собирает видео
   и текстовый пакет (`content.md`) вместе в `export_output/<таймстамп>/`.

### Быстрый запуск

Вместо `streamlit run app.py` вручную можно использовать `scripts/start_app.command`
(или ярлык на рабочем столе) — поднимает сервер, если он ещё не запущен, и открывает
браузер на `http://localhost:8501`.

## Ограничения

- **Trends MCP** — 100 запросов/месяц на бесплатном тарифе.
- **HeyGen** — рендер видео списывает премиум-кредиты вашего плана (Creator и выше).
  Точный расход зависит от длительности и не был точно измерен на момент написания —
  ориентируйтесь на остаток в `heygen auth status`, а не на фиксированную цифру.
- **Reddit-коллектор** требует доступа к `reddit.com/prefs/apps` для получения
  `REDDIT_CLIENT_ID`/`REDDIT_CLIENT_SECRET` — в некоторых сетях этот адрес заблокирован.
- **Голосовой ввод идеи** — распознавание речи идёт через бесплатный сервис Google
  (без ключа), нужен интернет и разрешение браузера на доступ к микрофону. Для
  коротких фраз работает достаточно точно, для длинных — может путать слова.
- **HeyGen Video Agent** по умолчанию воспринимает переданный текст как творческое
  задание и может переписать сценарий по-своему — в `video_generator.py` сценарий
  оборачивается явной инструкцией использовать текст дословно. Если агент всё равно
  своевольничает, доработать результат можно через `heygen video-agent create --mode chat`,
  не перегенерируя с нуля.
- Демо на защите — используйте заранее сгенерированный и сохранённый пример из
  `export_output/`, не полагайтесь на генерацию видео вживую (20-45 минут).
